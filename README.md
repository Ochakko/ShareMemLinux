# ShareMemLinux
simple share memory programs for Linux. using XSI shm and POSIX semaphore.

This programs are developed on CentOS 7.
So, licence is GPLv2.

make all

(at one console)
sudo ./sharemem_server

(at other console)
sudo ./sharemem_client

press Ctrl+C to quit
